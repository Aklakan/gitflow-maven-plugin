gitflow:release-finish

Full name:

com.amashchenko.maven.plugin:gitflow-maven-plugin:1.7.0:release-finish

Description:

The git flow release finish mojo.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
